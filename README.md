# Personal Screen-Time & Study Balance Tracker

## Language
Python

## GUI
Tkinter

## Purpose
Helps students track daily study hours vs screen time ethically.

## Features
- Daily input of study and screen time
- Balance calculation
- Feedback message
- Local data storage
- Summary view

## Ethical Aspect
- No internet usage
- No data sharing
- Local storage only
- Promotes self-discipline

## How to Run
```bash
python screen_time_tracker.py
